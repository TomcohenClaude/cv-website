# CLI-Style CV Website for Solution Engineer

A modern, interactive CV website featuring a terminal/CLI aesthetic, designed to showcase technical expertise and professional experience in a visually engaging way.

## Project Overview

This project presents a professional CV in a command-line interface format, combining retro terminal aesthetics with modern web technologies. The site is specifically tailored for a Solution Engineer role, emphasizing technical skills, problem-solving capabilities, and professional accomplishments.

## Features

### Visual Design
- **Terminal Interface**: Authentic CLI/terminal appearance with command prompts
- **Retro CRT Effects**: Scanline animations and phosphor glow effects
- **Matrix Rain Animation**: Subtle background animation for enhanced visual appeal
- **Color Scheme**: Classic green-on-black terminal colors with accent highlights
- **Responsive Design**: Optimized for desktop and mobile viewing

### Interactive Elements
- **Typing Animation**: Dynamic text rendering on page load
- **Hover Effects**: Glitch animations and color transitions
- **Smooth Scrolling**: Seamless navigation between sections
- **Easter Eggs**: Hidden interactions for engaged visitors

### Content Sections
- **Profile**: Contact information and professional identity
- **Skills**: Categorized technical competencies
- **Experience**: Detailed work history with achievements
- **Education**: Academic background
- **Projects**: Portfolio of technical projects

## Technology Stack

- **HTML5**: Semantic markup and structure
- **CSS3**: Advanced styling with animations and effects
- **JavaScript**: Interactive features and dynamic content
- **No Dependencies**: Pure vanilla web technologies (no frameworks required)

## Getting Started

1. **Clone or Download**: Get the project files
2. **Open**: Simply open `index.html` in a modern web browser
3. **Customize**: Update the placeholder content with your actual CV information
4. **Deploy**: Host on GitHub Pages, Netlify, Vercel, or any static hosting service

## Customization Guide

### Updating Content
Edit `index.html` to replace placeholder text with your actual information:
- Personal details and contact information
- Technical skills and competencies
- Work experience and achievements
- Education and certifications
- Project portfolio

### Styling Adjustments
Modify `styles.css` to customize:
- Color scheme (change `#00ff00` to your preferred color)
- Font family and sizes
- Animation speeds and effects
- Layout spacing and dimensions

### Adding Features
Extend `script.js` to add:
- Command-line simulator functionality
- Interactive project demos
- Dynamic content loading
- Analytics tracking

## Deployment Options

### GitHub Pages
```bash

git branch -M main
git remote add origin <your-repo-url>
git push -u origin main
```
Enable GitHub Pages in repository settings.

### Netlify
Drag and drop the project folder to Netlify's web interface.

### Vercel
```bash
vercel deploy
```
